## Aplicação CRUD de linha de comando desenvolvida em C#, ADO.NET e SQLite

O propósito é ser uma aplicação onde o usuário possa registrar suas sessões de codificação, informando manualmente sua data e hora de início e término com o tempo de duração sendo calculado automaticamente.

### O que é possível fazer?

- Exibir uma tabela com todas as sessões
- Adicionar uma nova sessão
- Editar uma sessão existente
- Deletar uma sessão existente
